🔗 **Aceder ao projeto:**
https://wilmilabraga.github.io/nutricao/

## 📌 Objetivo

Criar uma plataforma simples e acessível para disponibilizar conteúdos de apoio ao acompanhamento nutricional e promover educação alimentar.

---

## ✨ Funcionalidades

* Navegação simples e intuitiva
* Conteúdo informativo para consultas
* Layout responsivo
* Estrutura leve e organizada


Abra o ficheiro `index.html` no navegador.
